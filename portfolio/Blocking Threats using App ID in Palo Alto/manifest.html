Evidence title: Blocking Threats using App ID in Palo Alto

Evidence description:

In this lab I configured a Palo Alto next‑generation firewall to identify applications with App‑ID instead of only using ports. I created and tested security policies that allow trusted applications and block unwanted traffic, and then analysed the traffic and threat logs to verify that the firewall correctly detected and blocked attacks and risky applications. This gave me hands‑on experience with application‑aware security and fine‑tuning firewall rules.
