Evidence title: WPL: configurations

Evidence description:

During my internship at CERcuits, I made backups of the network configurations to ensure that changes could always be recovered. I first stored the configuration files locally on my PC in a separate folder so I could work on them efficiently and keep them well organized. I also uploaded these files to Confluence so that a central and shared version of the documentation was always available.

Whenever I changed something on the switches, I updated the configuration immediately and saved it again. In this way, everything stayed up to date and it was always possible to return to a previous version if needed. This helped keep the network configuration safe, structured, and well documented.
