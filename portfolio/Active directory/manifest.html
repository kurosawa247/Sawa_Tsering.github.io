Evidence title: Active directory

Evidence description:

I chose this evidence because I invested a lot of time in understanding and setting up Active Directory. It was the first time I configured a full AD environment myself, and I learned a lot from solving problems independently. This project helped me grow as an IT supporter.

The assignment was to install and configure an Active Directory environment. I had to set up a domain controller, create users and groups, manage permissions and policies, organize OUs, and perform basic support tasks such as resetting passwords. From this, I learned how a Windows network environment is structured and how to support users more effectively.

This evidence shows that I can support users, handle registration questions, listen actively to their problems, and help them with account and access issues. It also shows that I understand how to manage users, groups, and permissions in a professional IT environment.
