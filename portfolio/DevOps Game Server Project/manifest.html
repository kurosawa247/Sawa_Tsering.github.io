Evidence title: DevOps Game Server Project

Evidence description:

In a team we designed and built an automated Minecraft game server on AWS using DevOps tools. The code and configuration (Dockerfile, Jenkinsfile, Ansible playbooks) were stored in GitHub and triggered a CI/CD pipeline in Jenkins on every change. Jenkins built a Docker image, pushed it to Amazon ECR and used Ansible to deploy the container on an EC2 instance with persistent EBS storage and the correct security groups and ports. At the end of the project we could all join the same Minecraft world on the shared server, showing that our pipeline and infrastructure worked end‑to‑end.
