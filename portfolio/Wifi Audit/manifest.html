Evidence title: Wifi Audit

Evidence description:

For this project I performed a WiFi audit at Cipal Schaubroeck in Geel. The goal was to analyze the wireless network coverage and performance in a shared enterprise environment with multiple companies using the same infrastructure.

The survey was performed in a public corridor zone using professional measurement tools and a floor plan provided by the network administrator.

🔧 What I did (evidence of maintenance & optimization work)

  • Performed wireless signal measurements using NetSpot

  • Created a WiFi heatmap based on real measurements

  • Analyzed signal strength and coverage zones

  • Identified channel congestion in the 2.4 GHz band

  • Evaluated channel allocation strategy

  • Reviewed VLAN segmentation and client isolation

  • Studied controller-based AP management

  • Reviewed rogue AP protection (AirDefense)

  • Discussed redundancy and segmentation with network admin

📈 Optimization & maintenance insights

During the analysis I discovered that there is channel saturation in the 2.4 GHz spectrum. This creates interference and reduces performance. I identified this as an optimization opportunity through:

  • better channel spread

  • more 5 GHz usage

  • possible AP redistribution

I also learned how enterprise WiFi environments maintain security using:

  • client isolation

  • VLAN separation

  • firewall segmentation

  • centralized controller management
