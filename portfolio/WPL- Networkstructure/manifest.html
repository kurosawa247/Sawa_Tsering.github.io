Evidence title: WPL: Networkstructure

Evidence description:

During my internship at CERcuits, I helped improve the network structure. The original setup was fairly flat and less organized, which made the connections harder to manage. Together with four Aruba managed switches, we transformed the network into a more star-shaped topology, which improved both connectivity and manageability.

First, we drew the old network layout on paper and documented it in NetBox so we had a clear overview of the existing and desired structure. After that, we physically checked the patching areas and adjusted the cabling and connections to match the new design. This made the network more structured, more efficient, and easier to maintain.

This work shows that I can maintain and optimize wired and wireless network environments by applying patches and updates when needed, adapting to changing requirements, and making practical optimizations to improve performance and organization.
