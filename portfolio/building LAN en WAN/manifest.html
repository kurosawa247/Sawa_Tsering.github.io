Evidence title: building LAN en WAN

Evidence description:

Evidence item: Project Connecting and Securing Networks – Build WAN with OSPF

I chose this evidence item because it was one of the most technical and complete assignments of the semester. In this project, I designed, configured, and tested a large part of a network for the fictional SafeUS/SafeEU environment. The project included LAN and WAN setup, VLANs, router-on-a-stick, static routes, centralized DHCP, and OSPF.

I worked step by step and tested whether the different network components worked correctly together. When something did not function as expected, I identified the problem, adjusted the configuration, and tested again until it worked properly. I also documented the network design, IP plan, and configuration changes using Visio, Excel, and notes.

How this evidence matches the Builder role

This evidence shows that I can:

  • test installations and configurations according to a test plan.

  • recognize errors and solve them systematically.

  • document network designs, configurations, and test results clearly.

  • use tools to work in a structured and efficient way.

  • build a stable and reliable network solution.

Through this project, I applied my knowledge of routing, switching, and OSPF in a realistic network environment. It showed that I can build, test, and document a network in a structured way.
