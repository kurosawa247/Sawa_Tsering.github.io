Evidence title: WPL: Network Testing

Evidence description:

During my internship, I tested different parts of the infrastructure to make sure everything worked correctly before anything was applied to the real network. We created a small local sandbox environment where we could safely test major changes, including Proxmox updates, backups and restores, and the setup of a domain controller and file server. I also tested VLAN communication, DFS, and other network services to verify that each component behaved as expected.

During these tests, I encountered several issues, such as connectivity problems, backup performance limitations, and configuration problems in the test domain and file server environment.Some services required adjustments before they worked properly. When a problem occurred, I first identified the cause, then corrected the configuration and tested again until the issue was resolved.

I documented each test and its findings carefully so that the results could be reused later. I wrote down what was tested, what worked, what failed, and what changes were needed. This documentation made it easier to compare test results, follow the troubleshooting process, and safely apply the final changes to the production network.
