Bewijsstuk: AWS Project – Build Your VPC and Launch a Web Server

Waarom ik dit bewijsstuk gekozen heb:
Ik heb dit bewijsstuk gekozen omdat het mijn eerste volledige ervaring was met het opzetten van een cloudomgeving in AWS. Ik ben fier op dit project omdat ik zelf een virtueel netwerk (VPC) heb opgebouwd en een werkende webserver online heb gebracht, inclusief de nodige veiligheids- en toegangsinstellingen. Het project gaf me een goed inzicht in hoe systeembeheer in de cloud werkt, wat een belangrijke vaardigheid is binnen het moderne IT-landschap.

Wat de opdracht was en wat ik geleerd heb:
De opdracht bestond uit het bouwen van een Virtual Private Cloud (VPC) in AWS met de juiste subnetten, routing tables, internet gateways en beveiligingsgroepen. Vervolgens moest ik binnen deze VPC een EC2-instance lanceren waarop ik een webserver installeerde (Apache of Nginx) en configureerde zodat deze publiek toegankelijk was via HTTP.
Tijdens dit project heb ik geleerd:

  • Hoe cloudinfrastructuur verschilt van traditionele on-premise systemen.

  • Hoe netwerkcomponenten (subnets, gateways, route tables) in AWS samenwerken.

  • Hoe ik een server kan opzetten, configureren, testen en beveiligen.

  • Hoe ik gebruikersrechten en security groups kan beheren om toegang te regelen.
    Daarnaast heb ik ervaren hoe belangrijk het is om systematisch te werken en configuraties te documenteren, zodat de omgeving reproduceerbaar en onderhoudbaar blijft.

Hoe dit bewijsstuk de doelen van de rol ‘Systeembeheerder’ aantoont:
Dit bewijs toont dat ik in staat ben om IT-systemen te implementeren, configureren, beheren en beveiligen, zowel lokaal als in de cloud.
Het project sluit aan bij de belangrijkste doelen van de rol Systeembeheerder, namelijk:

  • Een stabiele en veilige IT-omgeving opzetten en beheren.

  • Systemen (zoals servers en netwerken) configureren volgens de vereisten van de organisatie.

  • Proactief omgaan met beveiliging, toegangsbeheer en beschikbaarheid.

  • Problemen analyseren en oplossen binnen een operationele infrastructuur.

Door deze opdracht heb ik niet alleen mijn technische kennis van AWS en netwerkinfrastructuur verdiept, maar ook mijn inzicht versterkt in hoe cloudresources efficiënt beheerd worden. Dit project bewijst dat ik als systeembeheerder in staat ben om een volledige omgeving van nul op te bouwen en te onderhouden in een professionele cloudomgeving.
