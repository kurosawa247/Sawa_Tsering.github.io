Evidence title: configuring zabbix/tftp

Evidence description:

In a small school network project, I implemented a monitoring solution using Zabbix to supervise network devices and servers.

I installed and configured the Zabbix server and connected host to the monitoring system. I set up agents and network checks to monitor availability, performance metrics, and system resources. The configuration included host setup, templates, triggers, and dashboards to visualize the status of the environment.

Through this project, I gained hands-on experience with network and system monitoring, alert configuration, metric collection, and troubleshooting based on monitoring data. The setup allowed us to detect outages and performance issues in our project network in real time.
