Evidence title: WPL: Documentation

Evidence description:

I kept the network infrastructure documentation in NetBox up to date and updated the data whenever the physical network structure changed. I also created a nomenclature page in Confluence to define clear naming rules for devices, cables, and network components. By maintaining this documentation carefully, the network remained well organized and could be handed over more easily to others. This work supported several important goals, including system documentation, configuration documentation, incident documentation, and changelogging.
