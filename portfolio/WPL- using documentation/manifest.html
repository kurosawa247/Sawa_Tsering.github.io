Evidence title: WPL: using documentation

Evidence description:

At CERcuits, we document the problems we encounter and the way we solve them so that this information can be reused later. When we had to update the Proxmox server from version 8 to 9 and the backup server from version 3 to 4, we first checked whether this procedure had already been done before. Since that was not the case, we consulted the official Proxmox documentation and carried out the update step by step. We also created a document describing all the steps clearly, so this method can be reused in the future.

This supports the following subgoals:

  • Standards and regulations.

  • Safe IT use.

  • Safety procedure.
