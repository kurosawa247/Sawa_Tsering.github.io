Evidence title: zabbix installatie en TFTP

Evidence description:

I chose this evidence because it was one of the tasks I learned the most from and felt proud of. Installing and configuring Zabbix and TFTP required accuracy and a good understanding of how network systems work. It was a challenge to combine monitoring with backup-related network management, but solving the issues step by step helped me grow in both knowledge and confidence.

The assignment had two parts. First, I installed and configured Zabbix to create a working monitoring environment. I learned how to add hosts, monitor network traffic and system resources, and set up triggers to detect problems early. Second, I installed and tested a TFTP server. This taught me how network devices can back up and restore configurations safely.

This proof shows that I can monitor network usage, support wired and wireless connectivity, keep network documentation up to date, and create backups of network configurations.
