Evidence title: IPv4 validation and reporting script

Evidence description:

In the scripting course I wrote a Python script that automatically checks large sets of IPv4 addresses stored in multiple folders and text files. The script iterates through all files, validates every octet of each address, and logs all errors (file name, line number, and position of the wrong octet) to a central resultaten.txt report. At the end it also generates a summary of the total number of errors per dataset. This project strengthened my skills in Python, file and folder handling, loops and basic input validation.
